# Gym Membership Manager

**Owner:** Owner
**Data Storage:** `gym.json` (saved locally, auto-loads on next run)
**Requirements:** Python 3.10+

---

## Features

- Add new member
  - Name
  - Phone number
  - Membership type (Monthly / Quarterly / Yearly)
  - Start date
  - Expiry date
- View all members
- Search member by phone number
- Show expired memberships
- Update member details
  - Change name / phone
  - Change membership type
  - Renew membership
- Shows **current local time** and **current date (DD/MM/YYYY)** on the screen
- Saves data to `gym.json` so it persists after closing

---

## Project Files

Your folder should look like this:
Folder/
│
├── `main.py`
└── `gym.json` (auto-created after you add your first member)
